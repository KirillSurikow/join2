Help
What is Join? Join is an application based on a Kanban Board, which can help companies to organize their tasks.
How to use it
1. Every task you create, appears on the board. Here you can access a detailed view on the task or edit it. According to the status of a task, you can drag a task to the box, which is intended to the status of the task (To Do, In progress, Awaiting Feedback and Done)
2. You can generelly create tasks in the Add task section, but you can also create tasks from the contact section or on the board. If you create a task from the the contact section, you can automatically assign the person to a task, you have selected on the contact section. If you create a task from the board, you can inject a task to a status box. For example you have started and completed a task on your own, but now you need Feedback. So you create task in the status box 'Awaiting Feedback'.
3. On the summary, you can see an brief overview of all tasks with the most important informaton, for example the next deadline.
